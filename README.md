# TextImageMaker

将文字制作成图片

这是一款将文字制作成图片的工具。它使您在facebook，微信，微博，qq，twitter，line ...上发布的内容更生动。

功能：
1.可以选择图片背景;
2.可以指定文字的风格：颜色等;
3.可以一键分享到社交平台，如朋友圈;
4.自定义文本位置，居中，居左和居右
5.应用程序的大小很小只有1M
6.操作简单
